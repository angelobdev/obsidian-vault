Esempi utilizzati a lezione per mostrare come svolgere il debugging di un'applicazione.

debugging_eclipse --> contiene package debugging:
					  - Counter.java e Main.java riguardano l'esempio della somma dei primi n numeri
					  - Esempio.java riguarda metodi sequenzaCrescente e inversoArray

debugging_intellij --> contiene classe AverageFinder.java per il calcolo della media di un insieme di numeri

Il codice contiene volutamente degli errori per permettere la dimostrazione dell'utilizzo dei tool di debugging.
